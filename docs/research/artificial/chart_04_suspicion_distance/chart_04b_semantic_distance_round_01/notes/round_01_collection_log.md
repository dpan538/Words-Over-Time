# Round 01 Collection Log

## Overview

This pass collected Chart 4B evidence for semantic distance around `artificial`.
The goal was not to design a chart but to prepare evidence for later visual placement.

## What was reused

- Existing Chart 4 baseline metadata from the main `processed` folder.
- Existing expanded corpus evidence from round 02.
- Existing pejoration evidence from round 03.
- Existing bridge leads and lexical authority work from round 04.

## What was newly checked

- Modern dictionary and learner dictionary entries for `artificial` and comparator terms.
- Historical dictionary checkpoints for `artificial`, `natural`, `real`, `genuine`, and `authentic`.
- A small collocation pass for `realistic artificial` and key `artificial vs X` phrase families.
- Earlier snippet anchors already collected for `artificial tears`, `artificial manners`, food coloring, and absence claims.

## Main outcome

The evidence supports treating `artificial` as a semantic trajectory.
Its center of gravity is still made and non natural.
It drifts toward imitation, simulation, and fake adjacentness in some contexts.
It should not be collapsed into `fake`.

## Main limitations

- This round is still dictionary led.
- Exact phrase collocations for many `artificial vs X` searches were sparse.
- `simulated` and `realistic` still need wider corpus checking.
- Some historical newspaper examples still need image level verification if quoted later.

# Words Over Time

A bold, minimal MVP for a curated historical word-frequency visualization website.

## Routes

- `/` - typographic home page
- `/words/forever` - demo detail page for "forever"
- `/about` - about and methodology

## Local Development

```bash
npm install
npm run dev -- -p 9000
```

Then open `http://localhost:9000`.

## Current Data State

The first demo word is `forever`. All frequency values, earliest-usage claims, and timeline visuals are placeholders pending corpus, dictionary, and scanned-book verification.
